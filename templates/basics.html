Github Basics

Creating an Account

To create a Github account, visit the Github website and click on the "Sign Up" button.

You will be prompted to enter your name, email address, and a password. Once you have entered this information, click on the "Create Account" button.

Creating a Repository

Once you have created a Github account, you can create a repository. A repository is a place where you can store your code.

To create a repository, click on the "New Repository" button on the Github homepage.

You will be prompted to enter a name for your repository. Once you have entered a name, click on the "Create Repository" button.

Pushing Code to a Repository

Once you have created a repository, you can push code to it. To do this, you will need to use the Git command line tool.

If you do not have Git installed, you can download it from the Git website.

Once you have Git installed, you can open a terminal window and navigate to the directory where your code is located.

Then, you can use the following command to push your code to your repository:

git push origin master

This command will push your code to the master branch of your repository.
